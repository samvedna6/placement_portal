# Smart Campus Placement Portal

A MERN Stack based placement portal for students and recruiters.

## Features

- Student Authentication
- Recruiter Authentication
- Job Posting
- Job Applications
- Application Tracking
- Resume Upload
- Interview Scheduling
- Automated Email Notifications

## Tech Stack

- MongoDB
- Express.js
- React.js
- Node.js
- Nodemailer
